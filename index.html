❓ Calendar Help Guide

🎯 Getting Started

This calendar helps our family coordinate calls with Grandma. Each person gets their own color so we can easily see who's calling when.

📱 On Mobile Devices

• Tap and hold to select a time slot
• Use the "Month" and "List" buttons to switch views
• The List view works best on smaller screens

🖥️ On Computers

• Click and drag to select your preferred time
• You can see the whole month at once
• Hover over appointments to see details

🔄 Recurring Calls

Use the recurring option to automatically book the same time slot every week for a month. Perfect for regular weekly chats!
